# HANDOFF — промпт для передачи проекта в новый чат

Один промпт, который передаёт весь контекст Panamera Studio любой модели
(Opus 4.8, Fable 5, новая сессия — неважно). **Скопируй блок ниже целиком и вставь
первым сообщением в каждый новый чат.** После него ставь свою задачу.

Если проект менялся — сначала попроси текущую модель обновить этот файл
(«обнови HANDOFF.md под текущее состояние проекта»), потом переноси.
