# chat-sync — общий журнал чатов Claude Code + Codex

Фоновая служба для мака: каждые 10 минут сама собирает **все локальные чаты
Claude Code и Codex** в один журнал (`~/ChatJournal`), а оба ассистента приучены
заглядывать туда в начале сессии. Вы просто создаёте чаты где удобно — руками
ничего экспортировать не нужно.

## Что это даёт

- Создали чат в Codex → через ≤10 минут Claude может его «видеть» (через журнал).
- Создали чат в Claude Code → то же самое для Codex.
- Два макбука: журнал синхронизируется между ними через **приватный**
  GitHub-репозиторий — на обоих маках виден общий список.
- Телефон: подключаетесь к маку удалённым доступом — там уже всё есть; сам
  журнал тоже можно смотреть на GitHub с телефона.

## Установка (на каждом маке, один раз)

1. Создайте на GitHub **приватный** репозиторий, например `chat-journal`
   (Private! — в журнале вся ваша переписка).
2. В терминале:

```bash
curl -fsSL https://raw.githubusercontent.com/alexautorental/panamera-studio/claude/cloud-code-account-sharing-mp1em2/tools/chat-sync/install.sh -o /tmp/cs-install.sh
bash /tmp/cs-install.sh https://github.com/alexautorental/chat-journal.git
```

Всё. Установщик поставит службу (launchd, каждые 10 минут), сделает первый
экспорт и добавит инструкцию в `~/.claude/CLAUDE.md` и `~/.codex/AGENTS.md`,
чтобы ассистенты сами читали журнал.

## Важные ограничения (честно)

- Чаты **не появятся в списке чатов** другого приложения — Anthropic и OpenAI
  не пускают друг друга в свои списки. Вторая модель видит содержимое через
  журнал и восстанавливает контекст — это максимум возможного.
- Служба видит только **локальные** сессии (файлы на диске мака):
  Claude Code — `~/.claude/projects`, Codex — `~/.codex/sessions`.
  Облачные чаты Claude (Remote / claude.ai/code) на диске не лежат — поэтому
  для этой схемы чаты Claude на компьютере создавайте в режиме **Local**.
- Пути источников можно поменять в `~/.chat-sync/config.json` (например, если
  ваш клиент Codex хранит сессии в другом месте).

## Файлы

| Файл | Назначение |
|---|---|
| `sync.py` | Экспортёр: jsonl-транскрипты → Markdown + INDEX.md + git push |
| `install.sh` | Установщик: служба launchd/cron, конфиг, инструкции ассистентам |

## Управление

```bash
python3 ~/.chat-sync/sync.py        # запустить синхронизацию вручную
tail -20 ~/.chat-sync/sync.log      # посмотреть лог службы
launchctl unload ~/Library/LaunchAgents/com.chat-sync.plist   # выключить
```
